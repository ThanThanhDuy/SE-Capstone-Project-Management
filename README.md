# Chú ý: 
### Trước khi push code cần phải check code, lệnh phía dưới chỉ check code
### không tự động fix (nên dùng)
```
yarn lint
```

### check toàn bộ code và tự động fix code (không nên dùng)
```
yarn lint:fix
```
